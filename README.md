# うめむら半蔵門内科歯科 ホームページ

## プロジェクト概要

うめむら半蔵門内科歯科の公式ホームページ。  
React (Vite) で構築。4つの構造レイアウト × 5つのカラーテーマで比較・選択可能なビルダー付き。

## デザインコンセプト

- **ロゴ**: 歯の形にUの字をあしらったSVGロゴ（グレー系）
- **背景**: 白〜シルバー基調（テーマで切替可能）
- **トーン**: 上品・洗練・プロフェッショナル
- **アイコン**: SVGストロークアイコン（プロ仕様）
- **写真**: URL指定 or ファイルアップロードで差し替え可能

## レイアウトパターン（4種）

| ID | 名称 | 特徴 |
|----|------|------|
| A | Classic Vertical | 左テキスト＋右写真2カラム → 縦スクロール |
| B | Full-Image Hero | 全面写真ヒーロー＋テキストオーバーレイ |
| C | Split Horizontal | 左右50:50分割（左写真・右テキスト）|
| D | Minimal Centered | 全要素中央配置のミニマルデザイン |

## ファイル構成

```
src/
├── App.jsx                 # メインアプリ（テーマ選択コントローラー）
├── main.jsx                # エントリーポイント
├── components/
│   ├── Logo.jsx            # SVGロゴコンポーネント
│   ├── Icons.jsx           # SVGアイコン集
│   ├── FadeIn.jsx          # スクロールフェードインアニメーション
│   ├── PhotoCarousel.jsx   # 写真カルーセル
│   ├── ImageEditor.jsx     # 写真管理パネル（URL入力/ファイル選択）
│   ├── SectionTitle.jsx    # セクション見出し
│   └── layouts/
│       ├── LayoutA.jsx     # Classic Vertical
│       ├── LayoutB.jsx     # Full-Image Hero
│       ├── LayoutC.jsx     # Split Horizontal
│       ├── LayoutD.jsx     # Minimal Centered
│       └── SharedSections.jsx  # 全レイアウト共通セクション群
├── config/
│   ├── themes.js           # カラーテーマ定義
│   ├── content.js          # テキスト・FAQ・エリア等のデータ
│   └── images.js           # デフォルト画像URL
└── styles/
    └── global.css          # グローバルスタイル
```
